# Menu pour plusieurs personnes

## Objectif : 

Créer un programme qui génère un menu aléatoire pour plusieurs personnes, calcule le total et applique une réduction si nécessaire, en utilisant tableaux multidimensionnels, boucles imbriquées et conditions.

## Consignes :

1. Créer un tableau $menu avec les catégories entrees, plats et desserts.
    - Chaque élément est un sous-tableau contenant nom et prix.

2. Créer un tableau $personnes avec plusieurs noms.

3. Pour chaque personne :

    - Sélectionner un plat aléatoire dans chaque catégorie.
    - Afficher le menu (nom + prix).
    - Calculer le total.
    - Si le total dépasse 20 €, appliquer une réduction de 10 %.

4. Afficher le total à payer pour chaque personne.
